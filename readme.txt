== Description ==

Kashiwazaki SEO Dynamic Dateは、WordPressサイトのコンテンツに動的な日付を簡単に挿入できるプラグインです。[ksdate]ショートコードを使用することで、自動的に更新される日付を表示し、コンテンツの鮮度を保つことができます。

= 主な機能 =

* **動的日付表示** - 現在の日付を自動的に表示
* **カスタムフォーマット** - 様々な日付形式に対応
* **相対日付計算** - 過去や未来の日付を相対的に指定可能
* **SEO最適化** - コンテンツの鮮度維持により検索順位向上
* **軽量実装** - 外部ライブラリ不要で高速動作
* **多言語対応** - 国際化対応済み

= 使用例 =

**基本的な使用:**
`[ksdate]` - デフォルト形式で現在の日付を表示
`[ksdate format="Y/m/d"]` - カスタム形式で日付を表示

**相対日付の使用:**
`[ksdate format="Y年" offset="-20y"]` - 20年前の年を表示
`[ksdate format="Y年m月" offset="-6m"]` - 6ヶ月前の年月を表示
`[ksdate format="Y/m/d" offset="+7d"]` - 7日後の日付を表示

= SEOに効果的な使い方 =

1. **コンテンツの更新日表示**
   最終更新日を動的に表示することで、検索エンジンにコンテンツの鮮度をアピール

2. **年度情報の自動更新**
   統計データや年次レポートの年度を自動更新

3. **イベント・キャンペーン情報**
   期間限定情報を動的に管理

4. **コピーライト表記**
   フッターの著作権年度を自動更新

== Installation ==

1. プラグインのZIPファイルをダウンロード
2. WordPress管理画面の「プラグイン」→「新規追加」→「プラグインのアップロード」からZIPファイルをアップロード
3. プラグインを有効化
4. 投稿や固定ページで`[ksdate]`ショートコードを使用開始

または

1. `/wp-content/plugins/`ディレクトリに`kashiwazaki-seo-dynamic-date`フォルダをアップロード
2. WordPress管理画面の「プラグイン」メニューからプラグインを有効化
3. 投稿や固定ページで`[ksdate]`ショートコードを使用開始

== Frequently Asked Questions ==

= ショートコードが動作しない場合は？ =

プラグインが有効化されているか確認してください。また、ショートコードの記述が正しいか確認してください。

= どのような日付形式が使用できますか？ =

PHPのdate()関数と同じ形式文字列が使用できます。例：
* `Y` - 4桁の年（2025）
* `m` - ゼロ付き2桁の月（01-12）
* `d` - ゼロ付き2桁の日（01-31）
* `H` - 24時間形式の時（00-23）
* `i` - 分（00-59）
* `s` - 秒（00-59）

= オフセット値の単位は？ =

以下の単位が使用できます：
* `y` - 年
* `m` - 月
* `w` - 週
* `d` - 日

例：`-20y`（20年前）、`+3d`（3日後）

= タイムゾーンはどのように処理されますか？ =

WordPressの設定（設定 > 一般 > タイムゾーン）に従います。

= キャッシュプラグインとの互換性は？ =

動的なコンテンツのため、ページキャッシュを使用している場合は、該当ページのキャッシュ除外設定が必要な場合があります。

= 複数のオフセットを組み合わせることはできますか？ =

現在のバージョンでは、1つのオフセット値のみサポートしています。複雑な計算が必要な場合は、日数に換算して指定してください。

== 使用可能なパラメータ ==

**format** (オプション)
デフォルト: `Y年m月d日`
説明: PHPのdate()関数形式の日付フォーマット

**offset** (オプション)
デフォルト: なし
説明: 相対的な日付オフセット（例：-20y, +3d, -6m, +2w）

== サポートされているオフセット ==

* 年: y（例：-20y = 20年前）
* 月: m（例：-6m = 6ヶ月前）
* 週: w（例：+2w = 2週間後）
* 日: d（例：+7d = 7日後）

== 実用的な使用例 ==

**最終更新日の表示:**
`<p>最終更新: [ksdate format="Y年m月d日"]</p>`

**統計データの年度表示:**
`<h2>[ksdate format="Y年" offset="-1y"]の売上データ</h2>`

**イベント告知:**
`<p>[ksdate format="m月d日" offset="+7d"]開催予定</p>`

**コピーライト表記:**
`<p>© [ksdate format="Y"] Your Company. All Rights Reserved.</p>`
